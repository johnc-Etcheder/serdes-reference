Channel Insertion Loss (S21)
Measured at receiver input · typical PCB copper trace + connector
112G
224G
Pass mask
112G: Nyquist at 26.5 GHz — typically 25–35 dB IL. CTLE must compensate this rolloff.
224G: Nyquist doubles to 53 GHz — same trace now shows 35–45 dB IL. Skin effect ∝√f adds ~3 dB and dielectric ∝f adds another 6 dB vs 112G.
Return Loss (S11) — Impedance Quality
Reflection from connectors and via stubs · target > 12 dB
112G
224G
Via stub notch: A 2mm via stub resonates at ~18 GHz (in 112G band). At 224G, a 1mm stub resonates at ~18 GHz — right in the signal band. All stubs must be back-drilled.
Target: Return loss > 12 dB across the full band (26.5 GHz for 112G, 53 GHz for 224G).
TDR — Time Domain Reflectometry
Impedance profile vs physical distance on PCB
Impedance
50Ω target
Reading a TDR: Step edge sent into channel. Reflections arrive back in time proportional to distance (≈15 cm/ns in FR4). Upward bump = inductive discontinuity (connector). Downward dip = capacitive discontinuity (via stub). Flat sections = good 50Ω trace.

224G eye
3 eyes for PAM4: Each eye represents one transition (L0↔L1, L1↔L2, L2↔L3). Ideal eye height = Vppd/6. ISI closes eye vertically. Jitter closes eye horizontally.
224G eye is smaller due to more channel loss, less CTLE headroom, tighter jitter budget (9.4 ps vs 18.8 ps).
SNR Budget Waterfall
How SNR margin is consumed stage by stage through the chain
112G
224G
Starting SNR is set by TX swing and channel loss. Each stage consumes margin: channel IL consumes most, then AFE NF, CTLE noise boost, FFE noise boost, ADC quantization.
224G starts with less and each stage costs more. The remaining margin before the KP4 cliff is much tighter.

112G: Typically achieves 10–15 cm reach on Megtron 6 at < 2.4×10⁻⁴ pre-FEC BER.

Crosstalk — FEXT vs Frequency
Far-end crosstalk between adjacent lanes · cannot be equalized
112G FEXT
224G FEXT
NEXT
Crosstalk worsens with frequency. At 224G (Nyquist 53 GHz), FEXT is significantly higher than at 112G because mutual capacitance coupling scales with frequency. Cannot be cancelled by EQ — must be controlled by PCB layout (spacing, stitching vias, reference planes).
FEC Correction Count Histogram
Corrections per codeword · key early warning metric
112G healthy
224G typical
Degraded link
Healthy link: Vast majority of codewords need 0 corrections. 112G links with good margin rarely exceed 2–3 corrections/CW.
224G typical: Slightly more corrections per codeword due to tighter margin. Monitor trend over time.
Degraded: Distribution shifts right toward 15. At 15 corrections = cliff edge.
VCO Phase Noise Spectrum
LC-tank VCO · 112G at 53 GHz vs 224G at 106 GHz
112G VCO @ 53G
224G VCO @ 106G
Phase noise degrades at higher frequency. LC-tank VCO at 106 GHz has ~6 dB worse phase noise than at 53 GHz for the same inductor Q. This directly translates to more sampling clock jitter at 224G, causing horizontal eye closure at the slicer.
CDR Jitter Tolerance (JTOL) Mask
Maximum tolerable sinusoidal jitter amplitude vs frequency
112G JTOL
224G JTOL
JTOL is measured in UI (unit intervals). At low frequencies (within CDR BW), the CDR tracks jitter — tolerance is high. Above CDR BW, CDR cannot track — tolerance drops sharply.
224G tolerance is tighter in absolute picoseconds because the UI is only 9.4 ps vs 18.8 ps.
ADC Output Histogram — VGA Gain States
Distribution of ADC output codes · diagnostic for VGA and clipping
Optimal
Too low gain
Clipping
Optimal: Bell-shaped distribution using most of ADC range without rail saturation.
Too low gain: Signal uses only center portion of ADC range. Quantization noise dominates.
Clipping: Codes pile up at max/min values (rails). Hard nonlinear distortion — unrecoverable. Check if > 0.1% of codes at max value.
Pulse Response — ISI Cursor Profile
Channel impulse response · shows PMR and ISI tap magnitudes
112G channel
224G channel
Main cursor h[0]: The actual transmitted symbol amplitude after the channel.
Postcursors h[1], h[2]...: ISI tails from previous symbols. PMR = h[1]/h[0]. 224G has larger relative postcursors (worse PMR) because the channel causes more dispersion at higher frequencies.
What to Do When You See These Charts
Insertion Loss Issues
  • S21 drops sharply at a specific frequency → via stub resonance — back-drill or blind vias
  • IL monotonically high across all frequencies → trace too long or bad PCB material — check tan δ, shorten route
  • IL spikes correlated with frequency → impedance ripple — check trace width consistency
  • Run COM simulation on measured S-params before powering up SerDes — know if channel passes first
Eye Diagram Issues
  • Eye closed vertically → run baud rate sweep: improves = ISI (fix EQ), flat = noise (NF, crosstalk)
  • Eye closed horizontally → CDR jitter or deterministic jitter source — check phase noise
  • Asymmetric eye heights → nonlinearity (DAC, VGA) or threshold miscalibration
  • Outer eyes smaller than inner → compression at high amplitude — check VGA gain
  • 224G eye will always look smaller — Q ≥ 4 is the target (not Q ≥ 5 like 112G)
FEC / BER Issues
  • FEC correction count rising → margin shrinking — debug before the cliff
  • BER plateaus at high baud but not low → noise limited — check AFE NF, crosstalk
  • BER improves at lower baud → ISI limited — optimize EQ chain (CTLE, FFE, DFE)
  • All 128 lanes degrade simultaneously → power droop or shared clock issue
  • Burst errors longer than 2–3 symbols → precoding disabled — check enable register
